No magic, no hand-waving: every formula is explained symbol by symbol, every claim is cited, and every number is computed for real — by hand first, then in code. You finish the book having trained your own networks from scratch and understanding exactly what happens inside a language model.

---

## Who it's for

Programmers and curious engineers who want a *true* mental model of neural networks — not analogies. If you can read code and aren't afraid of a little algebra, you're ready. Every concept is shown three ways: in **math**, in **Swift**, and in **Python**.

## What's inside

- **19 self-contained lessons**, short enough to finish in one sitting.
- **Interactive, self-checking quizzes** in every lesson — you always know if you got it.
- **Worked numbers everywhere** — no result is asserted without being computed in front of you.
- **Runnable code** — pure-Python networks you build by hand, then the same nets in PyTorch on Google Colab.
- **Light & dark themes**, fully **printable**, and **zero dependencies** — just HTML, works offline.

## Course structure

| Phase | Theme | Lessons |
|------:|-------|--------|
| **1** | One neuron — the atom of modern AI | 1–2 |
| **2** | Many neurons → a network | 3–5 |
| **3** | How a network learns: loss, gradients, backprop | 6–9 |
| **4** | Build your own — train networks from scratch | 10–12 |
| **5** | Under the hood of GPT: tokens, attention, the transformer | 13–19 |

The arc runs from `y = σ(Σ wⱼxⱼ + b)` all the way to pretraining, RLHF, temperature, and why models hallucinate — with no step left unexplained.

## Appendix

Reference pages, printable and one click away:

- **Python for the Swift Developer** — every Python construct in the book, explained through Swift.
- **Formulas Cheat Sheet** — all the course math on one page, with verified sanity-check numbers.

## How to read it

The book is a set of static HTML files — no build step, no install.

- **Locally:** open [`index.html`](index.html) in any browser.
- **Online:** publish the folder with GitHub Pages and share the link.

Start with [Lesson 1 — The Artificial Neuron](lessons/0001-the-artificial-neuron.html).
